Windows Checklist

For Windows 10 and Windows Server 2019

The Round 3 Windows Checklist:

  1. Thoroughly read the README and make notes.
  2. Run updates immediately. Turn on Windows Update if not working.
  3. Enable GodMode (https://www.bleepingcomputer.com/tutorials/enable-godmode-in-windows/)
  4. Answer the Forensic Questions. Do NOT proceed with any major fixes.
  5. Disable the Guest User and Admin User
  6. Perform User Auditing
  7. Go into “secpol.msc” and change the Password Policies, Account Lockout Policies, User Rights Assignments, and Audit Policies.
  8. Disable “Do not require CTRL + ALT + DEL” (services.msc)
  9. Check on the updates. Restart if needed.
  10. Disable services/features that are unsafe to the computer. Be sure not to disable critical services mentioned in the README.
  11. Scan User Files for Non-Related, music, hacking tools, or games.
  12. Disable File Sharing on the C Drive.
  13. Disable Network Sharing.
  14. Delete suspicious apps. (Programs and Features) (ex. WIRESHARK, VUSE, REAL PLAYER, TEAMVIEWER, DRIVER SUPPORT, SMART PC FIXER.)
  15. Remove any unnecessary startup processes.
  16. Enable Additional LSA Protection
  17. Enable Windows Firewall.
  18. Update Firefox.
  19. Go through Firefox settings.
  20. Check updates again if needed.
  21. Enable Windows Defender (use regedit to fix if not working) and download Avira.
  22. Check ports for malware. (https://www.bleepingcomputer.com/tutorials/how-malware-hides-as-a-service/)
  23. Make sure automatic apps are enabled.
  24. Make sure the computer goes to sleep when idle.
  25. Use Vulnerability Lists to help discover more vulnerabilities.
  26. Use the CyberPatriot website (https://www.uscyberpatriot.org/competition/training-materials/windows) to find more vulnerabilities.
