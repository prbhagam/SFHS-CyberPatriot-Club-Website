Ubuntu Checklist

For Ubuntu 18

The Round 3 Ubuntu Checklist:

  1. Thoroughly read the README and make noted
  2. Run updates first, before anything. Make sure they start automatically
  3. Perform User Auditing
  4. Disable the Guest User
  5. Secure the Root User
  6. Answer the Forensic Questions
  7. Enforce Password Requirements, Lockout Policies, and Audit Policies
  8. Enable Firewall
  9. Download AVG AntiVirus
  10. Delete Unwanted Applications
  11. Delete Prohibited Files
  12. Disable Bad Services
  13. Check services -- delete things like ftp (pure-ftpd)
  14. Make sure Computer Goes to sleep when idle.
  15. Try making the Security Level High
  16. Do anything the README specifically tells you to do.
  17. Make sure Firefox and other apps mentioned in the README are up to date.
  18. Use Vulnerability Lists to help discover more vulnerabilities.
  19. Use Windows Vulnerabilities and attempt to implement them into Ubuntu
  20. Use the CyberPatriot website (https://www.uscyberpatriot.org/competition/training-materials/training-modules) to find more vulnerabilities.
